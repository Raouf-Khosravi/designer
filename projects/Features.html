Ice Cube Tray – Parametric Plastic Mold Design

• Tools Used:Extrude, Shell, Linear Pattern, Draft

• Overview:Designed a 16-cavity ice cube tray intended for plastic injection molding. The part features draft angles for smooth mold ejection, uniform wall thickness for consistent cooling, and a compact cavity layout. The model is fully parametric, allowing quick adaptation for different shapes and tray sizes.

• Key Highlights:
   -   16 tapered compartments for easy ice release
   -   Drafted walls for moldability
   -   Parametric layout for scalable production
   -   Suitable for domestic and commercial use

Helical Gear – Industrial Transmission Component

• Tools Used:Helix Curve, Cut-Extrude, Circular Pattern

• Overview:Modeled a precision helical gear using defined gear parameters including pitch, pressure angle, and helix angle. The design ensures smoother and quieter engagement compared to spur gears, and is suitable for high-load transmission applications.

• Key Highlights:
   -   ISO-standard gear geometry
   -   Optimized for torque and reduced noise
   -   Helical profile created using helix and cut operations
   -   CAM-ready for CNC manufacturing

Axial Fan Blade – Optimized Airflow Geometry

• Tools Used:Loft, Fillet, Circular Pattern

• Overview:Designed a 6-blade axial fan for efficient airflow using aerodynamic blade shapes. The blades were created using lofted profiles to ensure smooth surfaces and ideal air guidance. This design is suited for HVAC systems, electronic cooling, or appliance fans.

• Key Highlights:
   -   Lofted blades with smooth curvature
   -   Symmetric and balanced geometry
   -   Parametric hub and blade configuration
   -   Prototype-ready for CFD and airflow simulation

Ball Bearing – Mechanical Precision Part

• Tools Used:Revolve, Cut-Revolve, Circular Pattern

• Overview:Modeled a detailed ball bearing including inner/outer races and precision balls. Focused on dimensional tolerances and symmetry for rotational accuracy, simulating real-world mechanical assemblies.

• Key Highlights:
   -   Radial ball bearing with symmetric groove profiles
   -   Parametric dimensions for standard sizing
   -   Suitable for motion simulation and mechanical linkage design
   -   Ready for manufacturing drawings and tolerancing
